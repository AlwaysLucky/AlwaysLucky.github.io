腾讯云 centOS 7.5 install LNMP

Posted on 2018-10-31

首先,设置centOS 7 EPEL repository

yum install epel-release

进入正文

一,安装nginx

yum install nginx

启动nginx

systemctl start nginx

测试nginx启动成功

http://yourIPAddress/

出现下图 则第一步成功

设置nginx开机启动

systemctl enable nginx

二,安装MariaDB(MySQL) 注意这里装mysql是不行的,原因自行查询

yum install mariadb-server mariadb

启动MariaDB

systemctl start mariadb

接下来,执行

mysql_secure_installation

下面会进行MariaDB的一些配置

  • Set root password? [Y/n] <– 是否设置root用户密码
  • Remove anonymous users? [Y/n] <– 是否删除匿名用户
  • Disallow root login remotely? [Y/n] <–是否禁止root远程登录
  • Remove test database and access to it? [Y/n] <– 是否删除test数据库
  • Reload privilege tables now? [Y/n] <– 是否重新加载权限表
  • … Success! 出现Success 成功

设置MariaDB开机启动

systemctl enable mariadb

安装php

yum install php php-mysql php-fpm

改写配置文件 /etc/php.ini

找到;cgi.fix_pathinfo=1, 改写为cgi.fix_pathinfo=0, 并去掉’;’

改写配置文件 /etc/php-fpm.d/www.conf

添加 listen = /run/php-fpm/www.sock , 网上大部分教程是添加/var/run/php-fpm/php-fpm.sock, 亲测不行
listen.owner = nobody nobody是你的系统用户名
listen.group = nobody nobody是你的系统用户名
listen.mode = 0666 去除前面的’;’
user = nginx
group = nginx

start PHP

systemctl start php-fpm

设置开机启动

systemctl enable php-fpm

配置NGIX处理PHP页面nginx.conf

配置文件是 /etc/nginx/nginx.conf 要在这里改 与老版本的不一样


  server {
        listen       80 default_server;
        listen       [::]:80 default_server;
        server_name  _;
        root         /usr/share/nginx/html;
        index  index.php index.html index.htm;

    # Load configuration files for the default server block.
    include /etc/nginx/default.d/*.conf;
    error_page 404 /404.html;
        location = /40x.html {
    }
    error_page 500 502 503 504 /50x.html;
        location = /50x.html {
    }
    location ~ \.php$ {
        try_files $uri =404;
        fastcgi_pass unix:/run/php-fpm/www.sock;
        fastcgi_index index.php;
        fastcgi_param SCRIPT_FILENAME $document_root$fastcgi_script_name;
        include fastcgi_params;
    }
}</code>

重启nginx

systemctl restart nginx

测试php成功 祝你好运
